# Brainworm
To deploy Brainworm, you will need to update the default credentials to Praxis, and the base url within the `AGENTS.md` file.

Finally, place it in the user directory, depending on the computer use agent you wish to infect, renaming the file to match the expected format. 
- `~/.claude/CLAUDE.md`
- `~/.codex/AGENTS.md`
- `~/.gemini/GEMINI.md`

You can also place it in a project root if you don't want it to run whenever you launch a CUA.
